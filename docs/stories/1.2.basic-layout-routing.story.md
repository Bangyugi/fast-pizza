# Story 1.2: Basic Application Layout & Routing

## Status
Draft

## Story
**As a** Customer,
**I want** to see a consistent application layout with a header and main content area,
**so that** I can easily understand the site structure and navigate my journey.

## Acceptance Criteria
1.  React Router is installed and configured to handle client-side navigation.
2.  A primary `AppLayout` component is created which renders a header and a main content area for pages.
3.  The header prominently displays the company name, "Fast React Pizza Co."
4.  A placeholder for a cart summary/link is present in the header.
5.  A basic home route (`/`) is configured to render content inside the `AppLayout`.

## Tasks / Subtasks
- [ ] Task 1: Install React Router DOM. (AC: 1)
  - [ ] Subtask 1.1: Run `npm install react-router-dom`.
- [ ] Task 2: Create `AppLayout` component. (AC: 2)
  - [ ] Subtask 2.1: Create `src/ui/AppLayout.jsx`.
  - [ ] Subtask 2.2: Implement basic structure: header and `Outlet` for main content.
  - [ ] Subtask 2.3: Add "Fast React Pizza Co." to the header. (AC: 3)
  - [ ] Subtask 2.4: Add a placeholder for cart summary/link to the header. (AC: 4)
  - [ ] Subtask 2.5: Apply basic Tailwind CSS for header and layout.
- [ ] Task 3: Configure routing in `App.jsx`. (AC: 1, 5)
  - [ ] Subtask 3.1: Import `createBrowserRouter` and `RouterProvider`.
  - [ ] Subtask 3.2: Define a `createBrowserRouter` instance.
  - [ ] Subtask 3.3: Set `AppLayout` as the `element` for the root route's children.
  - [ ] Subtask 3.4: Define a home route (`/`) to render `Home.jsx` within `AppLayout`.
  - [ ] Subtask 3.5: Wrap the application with `RouterProvider` in `App.jsx`.
- [ ] Task 4: Verify layout and routing.
  - [ ] Subtask 4.1: Run the application (`npm run dev`).
  - [ ] Subtask 4.2: Visually confirm the header, main content area, and correct rendering of the home page.
