# Story 1.3: Menu Data Fetching & Display

## Status
Draft

## Story
**As a** Customer,
**I want** to view the full list of available pizzas fetched from the menu,
**so that** I can decide what I want to order.

## Acceptance Criteria
1. A service function is created to fetch menu data from the "Fast React Pizza Co." API.
2. A new route (`/menu`) is created and linked from the home page.
3. The `/menu` route uses a React Router `loader` to fetch the menu data before the page renders.
4. The `Menu` component successfully receives and displays a list of `MenuItem` components, one for each pizza.
5. If the API fetch fails, a user-friendly error message is displayed on the page instead of the menu.

  - [ ] Subtask 1.4: Set the `loader` for the new route to the `loader` function.

### API Specifications
- The application will consume the `Fast React Pizza API`.
- **Endpoint**: `https://react-fast-pizza-api.jonas.io/api`
- **Usage**: The `getMenu` function within `src/services/apiRestaurant.js` handles fetching the menu data. This function should be used.
- [Source: docs/brownfield-architecture/data-models-and-apis.md, docs/brownfield-architecture/integration-points-and-external-dependencies.md]

### Component Specifications
- **`App.jsx`**: This file must be updated to include the routing configuration for the `/menu` page, including a `loader` function to fetch data before rendering. [Source: docs/brownfield-architecture/source-tree-and-module-organization.md#key-modules-and-their-purpose]
- **`Menu.jsx`**:
    - **Location**: `src/features/Menu/Menu.jsx`.
    - **Functionality**: This component will use the `useLoaderData` hook from React Router to access the menu data. It will then iterate over the list of pizzas and render a `MenuItem` component for each one.
    - **Data Loader**: It should export an `async function loader()` that calls the API service to fetch the menu data.
- **`MenuItem.jsx`**:
    - **Location**: `src/features/Menu/MenuItem.jsx`.
    - **Functionality**: This is a presentational component that will receive a `pizza` object as a prop from the `Menu` component. For this story, it only needs to display the pizza's name. Full implementation will be in Story 1.4.
- **`Home.jsx`**:
    - **Location**: `src/ui/Home.jsx`.
    - **Functionality**: Should be updated to include a `<Link to='/menu'>Menu</Link>` to allow users to navigate to the menu page.
