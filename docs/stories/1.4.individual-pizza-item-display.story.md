# Story 1.4: Individual Pizza Item Display

## Status

Done

## Story

**As a** Customer,
**I want** to see the details of each pizza, including its name, ingredients, price, and availability,
**so that** I can make an informed choice about my order.

## Acceptance Criteria

1.  The `MenuItem` component correctly renders the pizza's image, name, ingredients, and unit price.
2.  If a pizza is marked as "sold out," its appearance is visually distinct (e.g., greyed out) and the price is replaced with the text "Sold Out."
3.  A non-functional "Add to cart" button is displayed on all available pizza items. This button will have no action in this story.

## Dev Notes

### Previous Story Insights

Story 1.3 (`1.3.menu-data-fetching-display.story.md`) established the `Menu` component's ability to fetch and pass menu data to `MenuItem` components. The detailed rendering of individual pizza items was explicitly deferred to this story (1.4). The `MenuItem.jsx` file was created/updated in Story 1.3 but its implementation for individual item display was not completed.

### Data Models

No client-side data models are explicitly defined; data structures are implicitly defined by the responses from the Fast React Pizza API. The `MenuItem` component will receive a pizza object with properties such as: `id`, `name`, `ingredients`, `unitPrice`, `imageUrl`, `soldOut`.
[Source: docs/brownfield-architecture/data-models-and-apis.md#data-models]
[Source: docs/brownfield-architecture/integration-points-and-external-dependencies.md#external-services]

### API Specifications

The `MenuItem` component will consume data that has been fetched from the Fast React Pizza API via `src/services/apiRestaurant.js`. No direct API calls are made from this component. The relevant endpoint is `https://react-fast-pizza-api.jonas.io/api/menu` which provides a list of pizza objects.
[Source: docs/brownfield-architecture/data-models-and-apis.md#api-specifications]
[Source: docs/brownfield-architecture/integration-points-and-external-dependencies.md#external-services]

### Component Specifications

- `src/features/Menu/MenuItem.jsx`: This component is responsible for displaying the details of an individual pizza item, including its image, name, ingredients, unit price, and availability status. It should visually differentiate sold-out items and include a non-functional "Add to cart" button for available items.
